# 프로젝트 소개
- 1980년 ~ 2024년도까지의 USD/KRW, JPY/KRW의 환율 데이터 분석
- 통화 환율에 대한 급등 급락 시각화
- 분기별 연도별 그래프 분류 및 최대치 최소치 분류


## 팀원
- 김기석
- 이유진
- 송현석

## 개발스택
### Python
- BeautifulSoup4
- Requests
- Pandas
- Plotylit
- Streamlit

## 서비스
- 통화 선택 (USD/KRW, JPY/KRW)
- 분류 기준 선택 (연도별, 분기별)
- 기간 선택 (1980/01/04 ~ 2024/06/30)
- 선택 기간 내 환율 최고치 및 최소치 출력
